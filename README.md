## 🔧 Getting Started

### Prerequisites
- [Clarinet](https://github.com/hirosystems/clarinet) - Clarity development environment
- Node.js 16+ - For running tests
- Git - Version control

### Installation
```bash
git clone https://github.com/mac/interdimensional-practice-portal.git
cd interdimensional-practice-portal
npm install
```

### Development
```bash
# Check contract syntax
clarinet check

# Run tests
npm test

# Start local devnet
clarinet integrate
```

## 🧪 Testing
